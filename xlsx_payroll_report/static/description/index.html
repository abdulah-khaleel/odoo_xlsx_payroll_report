Excel Payroll Report

A clean and simple excel payroll report, available for payslip batches.

Supports multiple Payslip Structures

If you have setup multiple structures to your payroll, the report will be dividied in separate worksheets, with payslips added to thier corresponding structure/worksheet. The xlsx payroll report can be generated for multiple payslip batches.

The report was built using the Base report xlsx app by ACSONE SA/NV , Creu Blanca , Odoo Community Association (OCA). This app was included as a dependency.
